## ✨ Core Platforms & Features (ওয়েবসাইটের বিশেষত্ব)
1. **Dark Luxury Theme (কসমিক স্ল্যাব ও নিয়ন গ্রিন থিম):** চমৎকার গ্লাস-মরফিজম এফেক্ট, স্মুথ মোশন এবং দৃষ্টিনন্দন নিয়ন গ্রিন বর্ডার ফিনিশিং ও ব্লার ওভারলে।
2. **Dynamic Title Routing (ডায়নামিক পেজ টাইটেল):** কোনো রাউট বা উইন্ডো স্ক্রল চেঞ্জ হলে ইন্টারসেকশন অবজার্ভার ব্যবহার করে ব্রাউজার ট্যাবের টাইটেল স্বয়ংক্রিয়ভাবে আপডেট হয়।
3. **Pristine Preloader (স্টাইলিশ লোডার):** সাইট পুরোপুরি লোড হওয়ার আগে পর্যন্ত ব্র্যান্ড রিলেটেড একটি নিয়ন পালসিং অ্যানিমেটেড স্পিনার লোডার স্ক্রিন থাকে।
4. **Vite + React (ফ্রন্ট-এন্ড):** সম্পূর্ণ টাইপ-সেফ, আল্ট্রা-ফাস্ট ইন্টারেক্টিভ কাস্টম উপাদান।
5. **Vercel Serverless Functions + Nodemailer (মেল ইঞ্জিন):** এক্সপ্রেস সার্ভার ছাড়াই সরাসরি ভার্সেল সার্ভারলেস ব্যাক-এন্ড আর্কিটেকচার জিমেইল এসএমটিপি ব্যবহার করে ক্লায়েন্টদের প্রজেক্ট ব্রিফ পাঠানো এবং অটো-রিপ্লাই মেল পাঠানো।
6. **Rate Limiting & Security (নিরাপত্তা):** প্রতি ঘন্টায় একই আইপি থেকে সর্বোচ্চ ৫টি রিকোয়েস্ট পাঠানোর রেট লিমিটার এবং CORS প্রটেকশন।

## 🚀 Local Installation & Run (কিভাবে লোকালি চালু করবেন)

### ১. ডিপেন্ডেন্সি ইন্সটলেশন
প্রজেক্টের রুট ফোল্ডারে টার্মিনাল ওপেন করে নিচের কমান্ডটি রান করুন:
```bash
npm install
```

### ২. এনভায়রনমেন্ট সেটআপ (.env)
ফাইল ডিরেক্টরিতে একটি `.env` ফাইল তৈরি করুন এবং আপনার জিমেইল অ্যাপ পাসওয়ার্ড সেট করুন (ডিটেইলস জানতে ও সহজে অ্যাপ পাসওয়ার্ড বের করতে `SETUP.md` ফাইলটি দেখুন):
```env
EMAIL_USER=khabbab.dev@gmail.com
EMAIL_PASS=আপনার_১৬_অক্ষরের_গুগল_অ্যাপ_পাসওয়ার্ড
PORT=3000
```

### ৩. ডেভেলপমেন্ট মোডে রান করা
```bash
npm run dev
```
সার্ভারটি লোকালি `http://localhost:3000` এ রান হবে এবং ফ্রন্ট-এন্ড ও ব্যাক-এন্ড একই সাথে কাজ করবে।

## ☁️ Vercel Deployment & Serverless Integration
আমরা প্রজেক্টে `vercel.json` এবং `api/send-email.js` দিয়ে চমৎকার ভাবে ভার্সেল রেডি বান্ডেল তৈরি করে রেখেছি।

### Vercel এনভায়রনমেন্ট ভ্যারিয়েবল সেটআপ গাইড:
১. **Vercel Dashboard**-এ যান এবং আপনার প্রোজেক্ট নির্বাচন করুন।
২. **Settings** → **Environment Variables** ট্যাবে প্রবেশ করুন।
৩. প্রথম ভ্যারিয়েবলটি যুক্ত করুন:
   - **Name:** `EMAIL_USER`
   - **Value:** `khabbab.dev@gmail.com`
৪. দ্বিতীয় ভ্যারিয়েবলটি যুক্ত করুন:
   - **Name:** `EMAIL_PASS`
   - **Value:** `আপনার ১৬ অক্ষরের গুগল অ্যাপ পাসওয়ার্ড` (যেমন: `ekhh lscb fpjs kupu`)
৫. **Save** বাটনে ক্লিক করুন এবং প্রোজেক্টটি পুনরায় **Redeploy** করুন।
