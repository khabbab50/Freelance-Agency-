# Setup Instructions (ইমেইল ফাংশনালিটি সেটআপ গাইড)

আপনার এজেন্সি ওয়েবসাইটে ইমেইল সিস্টেম সঠিকভাবে কাজ করানোর জন্য নিচের পদক্ষেপগুলো অনুসরণ করুন:

---

### Step 1: Get Google App Password (গুগল অ্যাপ পাসওয়ার্ড নিন)

নিরাপত্তার জন্য গুগল সরাসরি জিমেইল পাসওয়ার্ড ব্যবহার করতে দেয় না। তাই আপনাকে একটি অ্যাপ পাসওয়ার্ড তৈরি করতে হবে:
1. আপনার **Google Account** (https://myaccount.google.com) এ লগইন করুন।
2. বাম পাশের মেনু থেকে **Security** ট্যাবে ক্লিক করুন।
3. **2-Step Verification** চালু না থাকলে তা চালু করে নিন।
4. **2-Step Verification** সংলগ্ন অপশনে ঢুকলে একদম নিচের দিকে **App passwords** অপশন পাবেন।
5. সেখানে নাম দিন `Freelance Agency Website` এবং **Create** বাটনে ক্লিক করুন।
6. উইন্ডোতে ১৬ অক্ষরের একটি কোড (পাসওয়ার্ড) পাবেন। এটি কপি করে নিন।

---

### Step 2: Configure Environment Variables (এনভায়রনমেন্ট কনফিগার করুন)

আপনার কোডের রুট ডিরেক্টরিতে `.env` নামে একটি ফাইল তৈরি করুন (যদি না থাকে) এবং নিচের মতো করে কপি করা পাসওয়ার্ডটি বসিয়ে দিন:

```env
EMAIL_USER=khabbab.dev@gmail.com
EMAIL_PASS=আপনার_কপি_করা_১৬_অক্ষরের_পাসওয়ার্ড_এখানে_বসাবেন
PORT=3000
```

*(দ্রষ্টব্য: পাসওয়ার্ডের মধ্যে কোনো স্পেস রাখবেন না)*

---

### Step 3: Install Dependencies & Start (ডিপেন্ডেন্সি ইনস্টল ও রান করুন)

মেইল সার্ভারটি লোকালি চালু করার জন্য নিচের কমান্ডগুলো রান করুন:

1. **ডিপেন্ডেন্সি ইনস্টল করতে:**
   ```bash
   npm install
   ```

2. **সার্ভার রান করতে:**
   ```bash
   npm run dev
   ```

সার্ভারটি লোকালি `http://localhost:3000` এ রান হবে এবং ফ্রন্ট-এন্ড ফর্ম সাবমিশনের সাথে সাথে আপনার মেইলে চমৎকার ডিজাইনের ইমেইল চলে যাবে।

---

### Step 4: Pro-tips & Troubleshooting (টিপস এবং সমস্যা সমাধান)
- **রেট লিমিটিং (Rate Limiting):** স্প্যামিং রোধ করতে একই আইপি অ্যাড্রেস থেকে প্রতি ঘন্টায় সর্বোচ্চ ৫টি মেসেজ পাঠানো যাবে।
- **ডিবাগিং (Debugging):** ইমেইল পাঠানো কিংবা ফেইলার সম্পর্কিত সকল ঘটনা দেখতে আপনার টার্মিনাল/কনসোল লগ খেয়াল করুন।
